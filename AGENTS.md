Проект состоит из web приложения, android и iOS приложения.
Сервер на nodejs. Сервер хранит данные в памяти и не использует БД (для демо целей)

UI приложения daily do. 
3 экрана:
- список активнойстей по дням, сверху отображаются самые новые
- экран создания списка задач  со след полями: имя, список задач 
- экран просмотра редактирования прогресса по выбранному дню, напротив каждой задачи должен быть checkbox, экран разделен на невыполненные и выполненных задачи. при клике на checkbox задача перечеркивается и отправляется вниз списка в раздел выполненных
