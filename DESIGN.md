## Экраны

### 1. Список активностей (главный экран)

- Список карточек, отсортированных по дате — новые записи сверху
- Каждая карточка: дата, название списка, краткий прогресс (например, `3 / 5 задач`)
- FAB-кнопка (Primary) в правом нижнем углу — создать новый список
- Фон экрана: **Background**

### 2. Создание списка задач

- Поле: **Имя списка** — текстовый ввод
- Поле: **Задачи** — многострочный список, кнопка «+ Добавить задачу»
- Кнопка «Сохранить» — цвет Primary, полная ширина
- Фон экрана: **Background**

### 3. Просмотр / редактирование прогресса

Экран разделён на две секции:

#### Секция «Невыполненные»
- Чекбокс: контурный, цвет Primary
- Текст задачи: **Text Primary**

#### Секция «Выполненные»
- Чекбокс: заполненный, цвет **Checkbox Done** (`#34C759` / `#30D158`)
- Текст задачи: зачёркнутый, цвет **Text Done** (светлее основного в светлой теме, темнее — в тёмной)
- При нажатии на чекбокс задача анимированно перемещается между секциями

## Компоненты

### Чекбокс задачи

```
Невыполненная:  [ ]  Текст задачи          ← контур Primary
Выполненная:   [✓]  ~~Текст задачи~~       ← заливка зелёная, текст Text Done
```

### Карточка дня

```
┌─────────────────────────────────┐
│  Пн, 27 марта              3/5  │  ← Text Secondary / Label Small
│  Название списка                │  ← Title Medium / Text Primary
└─────────────────────────────────┘
```
Фон карточки: **Surface**. Скругление углов: 12 dp. Тень: лёгкая (elevation 2).

### Кнопка (Primary)

- Фон: **Primary**
- Текст: белый (`#FFFFFF`)
- Скругление: 8 dp
- Состояние Disabled: opacity 40%

## Прочее

- **Иконки**: Material Symbols (outlined)
- **Скругления**: карточки — 12 dp, кнопки — 8 dp, чекбоксы — 4 dp
- **Отступы**: базовая сетка 8 dp
- **Анимации**: перемещение задачи между секциями — `animateItemPlacement` (300 ms, EaseInOut)
